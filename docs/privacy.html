Politique de Confidentialité

Application Webshort — Dernière mise à jour : 11 février 2026

1. Introduction

Webshort est une application de création et publication automatisée de Reels Instagram à partir de snippets de code. Cette politique décrit comment nous collectons, utilisons et protégeons vos données.

2. Données collectées

Lorsque vous utilisez Webshort, nous pouvons collecter :

  • Données d'authentification Instagram : token d'accès et identifiant de compte Instagram Business, nécessaires pour publier du contenu en votre nom.
  • Contenu créé : les snippets de code, titres et hashtags que vous saisissez pour générer des Reels.
  • Données techniques : logs de traitement (horodatages, statuts des jobs) stockés temporairement en mémoire.

3. Utilisation des données

Vos données sont utilisées exclusivement pour :

  • Générer des vidéos Reels à partir de vos snippets de code
  • Publier du contenu sur votre compte Instagram avec votre autorisation
  • Planifier des publications futures selon vos paramètres

4. Stockage et sécurité

Les données sont traitées localement sur votre machine. Les tokens d'accès Instagram sont stockés dans des fichiers de configuration locaux et ne sont pas transmis à des tiers. Les vidéos générées sont stockées localement dans le dossier de sortie de l'application.

5. Partage des données

Nous ne vendons, ne louons et ne partageons pas vos données personnelles avec des tiers. Les seules communications externes sont :

  • Les appels à l'API Instagram Graph pour la publication de contenu (à votre demande)
  • Les appels à l'API Anthropic Claude pour la génération de contenu (si activé)

6. Vos droits

Vous pouvez à tout moment :

  • Révoquer l'accès de l'application à votre compte Instagram via les paramètres Instagram
  • Supprimer vos données locales en supprimant le dossier workspace de l'application
  • Demander la suppression de vos données (voir notre page de suppression des données)

7. Cookies

L'application n'utilise pas de cookies. Aucun suivi publicitaire n'est effectué.

8. Suppression des données

Pour supprimer vos données, consultez notre page dédiée à la suppression des données.
