Suppression des Données

Application Webshort — Dernière mise à jour : 11 février 2026

Comment supprimer vos données

Webshort traite vos données localement sur votre machine. Voici comment supprimer l'ensemble de vos données :

1Révoquer l'accès Instagram

Allez dans vos paramètres Instagram → Sécurité → Applications et sites web → Supprimez l'accès de "Webshort".

2Supprimer les vidéos générées

Supprimez le dossier out/ dans le répertoire de l'application. Ce dossier contient toutes les vidéos brutes et finales générées.

3Supprimer les espaces de travail

Supprimez le dossier workspace/ dans le répertoire de l'application. Ce dossier contient les fichiers HTML temporaires et les snippets de code.

4Supprimer la configuration

Supprimez le fichier .env et config.yaml qui contiennent vos tokens d'accès et paramètres.

Données stockées par des tiers

  • Instagram / Meta : Les Reels publiés sur votre compte Instagram restent sur la plateforme. Vous pouvez les supprimer directement depuis Instagram.
  • Anthropic (Claude API) : Si vous utilisez la génération de code par IA, les requêtes sont traitées selon la politique de confidentialité d'Anthropic.

Confirmation

Une fois les étapes ci-dessus effectuées, toutes vos données auront été supprimées de l'application Webshort. Aucune donnée n'est conservée sur des serveurs distants par notre application.
