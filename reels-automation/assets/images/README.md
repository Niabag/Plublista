# Images Assets

Ce dossier contient les images partagées utilisables dans vos snippets de code.

## Comment utiliser

1. **Placez vos images ici** (ex: `logo.png`, `banner.jpg`, etc.)

2. **Dans votre code HTML**, référencez-les simplement par leur nom:
   ```html
   <img src="logo.png" alt="Logo">
   ```

3. Les images seront **automatiquement copiées** dans le workspace de chaque job lors de la création.

- Ajoutez d'autres images selon vos besoins
